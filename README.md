# 爱客服采集客户端

爱客服AI客服系统配套Windows桌面采集客户端。

负责采集拼多多店铺的聊天消息、订单、商品信息，推送到爱客服服务器进行AI分析处理，并自动发送AI回复。

## 功能特性

- 🏪 多店铺管理（添加/启动/停止）
- 💬 实时消息采集（文本/图片/订单卡片/商品咨询）
- 📦 订单自动采集与同步
- 🛍️ 商品信息采集同步知识库
- 🤖 AI回复自动发送
- 👨‍💼 一键切换人工模式
- 📊 数据统计面板
- 🔔 桌面通知提醒
- 🔒 账号��码AES加密存储
- 📦 支持打包成Windows EXE安装包

## 技术架构

- **UI**: PyQt6 + PyQt6-Fluent-Widgets
- **采集**: Playwright + WebSocket直连拼多多
- **数据库**: SQLite本地缓存
- **服务器**: 对接爱客服服务器 8.145.43.255:6000
- **打包**: PyInstaller + Inno Setup
